🃏 WBD 프로필 등록기

게임 로그에서 선수 추출 → 사진 매핑 → profile_map.js 내보내기
📖 사용 방법
1. 게임 로그 .txt 파일 드래그 (또는 선택) → 모든 선수 자동 추출
2. 사진 폴더 드래그 (또는 파일들 선택) → 선수명과 파일명 자동 매칭
3. 미매핑 선수는 직접 파일명 입력 또는 파일 선택으로 매핑
4. 💾 profile_map.js 내보내기 클릭 → 분석기 폴더에 저장
5. 분석기와 같은 폴더에 profile/ 폴더 만들고 사진들 넣기
📂 1단계: 게임 로그 불러오기
📄

🖼️ 2단계: 프로필 사진 불러오기
📁
사진 파일들을 드래그하거나 클릭해서 선택
폴더 통째로 드래그도 OK · 파일명이 선수명과 비슷하면 자동 매칭
